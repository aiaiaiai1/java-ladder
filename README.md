## 기능
- [ ] 사다리 생성 기능

## 입력
- [ ] 참여자 이름 입력
  - [x] 최대 5글자까지 가능
  - [ ] 공백인 경우 예외처리
  - [ ] 중복인 경우 예외처리
- [ ] 최대 사다리 높이 입력
  - [ ] 0인 경우 예외처리

## 출력
- [ ] 사다리 결과 출력
  - 사람 이름을 5자 기준으로 출력
  - 다리가 있는 경우 |-----|
  - 다리가 없는 경우 |     |
